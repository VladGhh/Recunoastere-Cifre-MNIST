\# Recunoaștere Cifre - Rețea Neuronală de la Zero



Acest proiect implementează o rețea neuronală artificială simplă pentru recunoașterea cifrelor scrise de mână (setul de date MNIST).



\## Caracteristici

\- Scris în Python folosind doar \*\*NumPy\*\*.

\- Implementează algoritmul \*\*Stochastic Gradient Descent\*\*.

\- Include o funcție pentru a testa propria imagine creată în Paint.



\## Instalare
